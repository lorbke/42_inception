Inception

A WordPress site rebuilt from nothing but Alpine, one service per container. This page is served by a different container than the rest of it — no PHP, no database, no framework.

lorbke

The stack

you are here static busybox httpd
nginx tls 1.2 / 1.3 · :443
wordpress php-fpm · :9000
mariadb :3306
redis object cache · :6379
ftp vsftpd · :21

Only nginx is reachable from outside. Everything else answers to its service name on an internal bridge network, and nothing else.

Notes

Base
alpine 3.23, pinned. Every image built from a hand-written Dockerfile.
Persistence
Two named volumes, backed onto the host. The database and the site files outlive every container.
Processes
One service per container, running in the foreground. Setup happens in an entrypoint, which then hands the PID 1 slot to the real process.
This page
Static HTML on busybox httpd. It shares nothing with the WordPress stack but the network.

Built for the 42 curriculum. Certificate is self-signed — that part is on purpose.
